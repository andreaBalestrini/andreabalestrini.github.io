ArduinoTube

ArduinoTube is a project, based on the ArduinoYun board, which I developed in 2014 as an intern for the municipality of Saint-Marcel. The aim of the project was to create a ROV useful for the inspection of the rain irrigation system pipes, in order to make maintenance easier and cheaper.
Through a web application it is possible to control the ROV by connecting to a special interface and, at the same time,  to see through an ipCam installed on board.
